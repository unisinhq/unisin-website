## Feature Roadmap

### Phase 1 — MVP (Launch)
Core unified inbox functionality.

Platforms:
- Email (Gmail, Outlook, IMAP)
- Instagram Business DMs
- LinkedIn Page messages

Features:
- Unified chronological inbox
- Platform indicators
- Read / reply functionality
- Labels (Brand, Collab, Fan, Follow-up)
- Search and filtering
- Basic auto-replies
- Solo creator accounts

---

### Phase 2 — Expansion
Broader platform coverage and workflow tools.

Platforms:
- Discord (via bot)
- X (Twitter) DMs (read-first)
- Reddit inbox / modmail
- YouTube comments & inquiries

Features:
- Saved replies
- Assistant access
- Conversation assignment
- Notification controls
- Message analytics

---

### Phase 3 — Monetization & Ops
Move beyond inbox into creator operations.

Features:
- Deal tagging & tracking
- Brand profiles
- Revenue attribution
- Calendar integration
- Agency dashboards
- SLA / response-time metrics

---

### Phase 4 — Platform
UNISIN becomes creator infrastructure.

Features:
- API access
- Brand-side portals
- Marketplace integrations
- Advanced automation (opt-in)

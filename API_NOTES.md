## API Notes & Platform Integrations

UNISIN integrates exclusively through official APIs and approved access.

---

### Email
**Providers**
- Gmail API
- Outlook / Microsoft Graph
- IMAP fallback

**Capabilities**
- Read messages
- Send replies
- Attachments
- Threading

**Notes**
- Backbone of reliability
- Primary channel for brand deals

---

### Instagram (Business Accounts Only)
**API**
- Meta Graph API
- Instagram Messaging API

**Capabilities**
- Read DMs
- Send replies
- Webhooks for real-time updates

**Restrictions**
- Business or Creator accounts only
- No personal accounts
- Requires Meta app approval

---

### LinkedIn
**API**
- LinkedIn Marketing / Pages API

**Capabilities**
- Read Page inbox messages
- Reply as Page admin

**Restrictions**
- Personal DMs not supported

---

### Discord
**Method**
- Bot-based integration

**Capabilities**
- Read messages from configured channels
- Reply via bot identity
- Server-specific permissions

**Notes**
- Ideal for creator communities and brand outreach servers

---

### X (Twitter)
**API**
- Paid API tiers
- DM access limited

**Capabilities**
- Message ingestion
- Reply via deep link or native client

---

### Reddit
**API**
- OAuth-based Reddit API

**Capabilities**
- Inbox messages
- Modmail
- Reply support (rate-limited)

---

### YouTube
**API**
- YouTube Data API

**Capabilities**
- Comment ingestion
- Channel business inquiries

**Limitations**
- No native DM inbox equivalent
